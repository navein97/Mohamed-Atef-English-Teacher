### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Node.js (for running a local server)

### Running the Website

#### Option 1: Using Live Server (Recommended)

1. Install live-server globally (if not already installed):
   ```bash
   npm install -g live-server
   ```

2. Navigate to the project directory:
   ```bash
   cd "c:\Users\ACER\Kuwait\Mohamed Atef English Teacher"
   ```

3. Start the server:
   ```bash
   npm start
   ```
   or
   ```bash
   live-server .
   ```

4. The website will automatically open in your browser at `http://127.0.0.1:8080`

#### Option 2: Direct File Opening

Simply double-click `index.html` to open it in your default browser. Note: Some features may work better with a local server.

#### Option 3: Using Python

If you have Python installed:

```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```
